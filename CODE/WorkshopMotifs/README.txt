# Course Workshops Repository

This repository contains the documentation and code for all the course workshops. Each workshop has its own folder with a summary of the workshop development, the contents of the folder, and the report and code.

## Workshops

1. **Motif Finder**
   - Description: A Java program to find motifs in DNA sequences and calculate the sequence with the highest entropy.
   - Contents: Code

2. **Perform Experiments**
   - Description: A Java program to perform experiments on motif finding with different parameters and save the results in a CSV file.
   - Contents: Code
